# 💼 AI Interview Practice Bot

An AI-powered interview preparation platform that helps users practice real-time interviews by generating role-based questions, evaluating responses, and providing instant feedback.

## 🚀 Features

- 🎯 Role-based interview questions (Developer, Data Scientist, HR, etc.)
- 📊 Instant answer evaluation with score and feedback
- 🧠 AI-generated personalized interview experience
- 🎤 Supports voice input (via device voice typing)
- 📱 Mobile-friendly and responsive design
- ⚡ Fallback system to handle API limits

## 🛠️ Tech Stack

- Python
- Streamlit
- Google Gemini API
- HTML/CSS (via Streamlit UI)

## 💡 How It Works

1. Select a job role and difficulty level
2. Generate an interview question using AI
3. Answer using text or voice typing
4. Receive AI-based evaluation with score and improvement tips

## 🎯 Goal

To simulate a real interview environment and help users improve their communication, technical knowledge, and confidence.

## 📌 Future Enhancements

- Multi-round interviews
- Timer-based sessions
- Resume-based question generation
- Performance analytics dashboard
